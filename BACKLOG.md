# CoachBoard Pro - Backlog

Ideas and features recorded here. NOT implemented yet. Each item will be evaluated and scheduled in a future phase.

**Priority order (foundation first):**
1. Phase 0: Rollback Foundation (Git, backup, baseline tests)
2. Phase 1: Render Authority (Engine.render() is the ONLY render point)
3. Phase 2: Dependency Cleanup (eliminate circular dependencies)
4. Phase 3: System Isolation (each system independent)
5. Phase 4: Foundation Tests
6. Phase 5: Cleanup (sw.js, unused files)
7. Phase 6+: Feature development
